Sankey Module für IP-Symcon

Installation:
1. Modulordner in 'modules/SankeyModule/' kopieren.
2. IP-Symcon Konsole -> Module neu laden.
3. Neue Instanz vom Typ 'SankeyModule' erstellen.
4. Konfigurieren: PV-Quellen, Batterie, Netz, Verbraucher.
5. WebPath überprüfen, index.html im Browser aufrufen.
6. In index.html <InstanceID> durch die InstanzID ersetzen.

Fertig!
